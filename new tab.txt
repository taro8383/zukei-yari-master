    Topic 2: 四捨五入（ししゃごにゅう） / Rounding Off (Shishagonyuu)

        Goal / めあて: Find the approximate number by rounding off. / 四捨五入をして、わかりやすい「がい数（およその数）」を作ろう。

        How it Works / やりかた: 0, 1, 2, 3, 4 mean round down (切り捨て). 5, 6, 7, 8, 9 mean round up (切り上げ). / 0, 1, 2, 3, 4 は切り捨て。5, 6, 7, 8, 9 は切り上げるよ。

        Real-Life / 生活の中でのつかいみち: Quickly estimating if you have enough money to buy multiple games. / ゲームをいくつか買うとき、お金が足りるかパッと計算するとき！

        Prompt format: "Round 34,560 to the nearest thousand. / 34560を四捨五入して、千の位までのがい数にしましょう。" (Expected input: 35000).

    Topic 3: 切り捨てと切り上げ / Rounding Down and Rounding Up

        Goal / めあて: Know when to strictly round down or round up in real situations. / どんな時に「切り捨て」や「切り上げ」を使うか考えよう。

        How it Works / やりかた: For shopping budgets, you usually round up to be safe. / お金を準備するときは、足りなくならないように「切り上げ」を使うことが多いよ。

        Exercise Generation Logic: Generate word problems requiring either strict round up or round down based on context.

        Prompt format 1 (Round Up): "Kei wants to buy 3 items that cost 420 yen each. He only has 1000-yen bills. How many 1000-yen bills does he need? / けいくんは420円のアイテムを3つ買います。1000円札だけで払うとき、1000円札は何枚いりますか？" (420 * 3 = 1260. Requires 2 bills. 2枚).

        Prompt format 2 (Round Down): "Kei has 2500 yen. A capsule toy costs 300 yen. How many can he buy? / けいくんは2500円持っています。300円のガチャガチャは何回まわせますか？" (2500 / 300 = 8.33. Round down to 8. 8回).
