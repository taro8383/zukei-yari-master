Tab 10: 変わり方調べ / Investigating Changes

Visual Aid Requirements:

    Dynamic Data Tables: Render HTML <table> elements to display relationships between two changing variables (e.g., Number of squares vs. Total perimeter). Top row represents variable 〇, bottom row represents variable △.

Topics & Pre-Exercise Explanations:

    Topic 1: 表を完成させる / Completing the Table

        Goal / めあて: Find the pattern in the table and fill in the missing numbers! / 表の数字のきまりを見つけて、空いているマスをうめよう！

        How it Works / やりかた: Look at how the bottom number changes when the top number goes up by 1. / 上の数が1増えると、下の数がどう変わるか注目しよう。

        Real-Life / 生活の中でのつかいみち: Figuring out how much damage you deal if you upgrade your sword 5 times! / 剣を5回強化したら、ダメージがどれくらい増えるか予測するとき！

        Exercise Generation Logic: Generate a simple linear relationship (e.g., y=3x or y=x+4). Provide a table with x values 1 through 5. Leave two y values blank with <input> fields.

    Topic 2: きまりを見つける / Finding the Rule

        Goal / めあて: Explain the secret rule of the table in words! / 表にかくされた「きまり」を言葉で説明しよう！

        Exercise Generation Logic: Generate a completed table.

        Prompt format: "When 〇 increases by 1, how does △ change? / 〇が1増えると、△はどうなりますか？" (Provide multiple-choice buttons: Increases by X, Decreases by X).

    Topic 3: 〇と△を使った式 / Writing the Equation

        Goal / めあて: Turn the table's rule into a powerful math equation! / 表のきまりを「〇」と「△」を使った計算の式に変えよう！

        How it Works / やりかた: If △ is always 4 times bigger than 〇, the equation is 〇 × 4 = △. / △がいつも〇の4倍なら、「〇 × 4 = △」という式になるよ。

        Exercise Generation Logic: Generate a table. Provide an equation builder UI where the user must select the correct operator (+,−,×,÷) and input the correct constant to complete the formula 〇 [operator] [input] = △.
